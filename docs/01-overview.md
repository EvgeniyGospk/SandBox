# Particula — Project Overview

> Статус: текущая кодовая база в этом репозитории — `apps/web` (frontend) и `packages/engine` (Rust/WASM).
> Мультиплеер, AI/эволюция, DSL и backend‑часть описаны как целевое видение (roadmap) и могут отсутствовать в коде.

## Описание проекта

**Particula** — веб-приложение для симуляции физики частиц в 2D пространстве. Вдохновлено мобильной игрой Sandbox и The Powder Toy, но с улучшенной физикой, современным интерфейсом и мультиплеером.

## Ключевые особенности

- **Реалистичная физика** — температура, давление, гравитация для каждой частицы
- **Электричество** — с сопротивлением и напряжением
- **Живые существа** — AI + эволюция с генетическими алгоритмами
- **Мультиплеер** — до 10 игроков в комнате + галерея работ
- **DSL редактор** — пользователи создают свои элементы
- **Кроссплатформенность** — десктоп и мобильные устройства

## Целевая аудитория

- Любители sandbox-игр
- Образовательный сектор (физика, химия)
- Творческие люди (создание сцен, арт)
- Разработчики (создание своих элементов через DSL)

## Бизнес-модель

**Freemium:**
- Бесплатно: базовые элементы, локальное сохранение
- Платно: продвинутые элементы, облачное хранилище, приватные комнаты

## Конкуренты

| Проект | Платформа | Преимущества Particula |
|--------|-----------|----------------------|
| The Powder Toy | Desktop | Веб, мультиплеер, современный UI |
| Powder Game | Web (Flash) | Технологии не устарели, мобильная версия |
| Sandspiel | Web | Больше элементов, мультиплеер, DSL |
| Sandbox (mobile) | Mobile | Веб-версия, точнее физика, мультиплеер |
