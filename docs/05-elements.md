# Элементы Particula

## Лимит ElementId

В движке `ElementId` хранится как `u8`, поэтому проект поддерживает **не более 256 элементов** (ID `0..255`).

## Матрица реакций

```
         │ Water │ Fire │ Lava │ Electricity
─────────┼───────┼──────┼──────┼─────────────
Wood     │   -   │Burn  │Burn  │     -
Oil      │Float  │Burn  │Burn  │     -
Metal    │   -   │  -   │Melt  │  Conduct
Stone    │   -   │  -   │  -   │     -
Water    │   -   │Steam │Steam │  Conduct
Fire     │ Out   │  -   │  -   │     -
```

## Цветовые вариации

Каждая частица получает случайное отклонение цвета:
- Hue: ±5°
- Saturation: ±10%
- Lightness: ±10%

Это создаёт визуально приятную текстуру.
